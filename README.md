# Shop Website 

> Basic website for selling products and items.

## Usage 

Rename  "config/config.env.env" to "config/config.env" and update setting/values to your own 

## Install Dependencies

``` 
npm install 
``` 

## Run App 
```
#run in developer mode 
npm run dev

#run in production mode
npm start 
```
